
# Rules
- NUNCA realizar un merge hacia la rama master (o producción) sin pedir y recibir primero autorización expresa del usuario.

- NUNCA realizar commits ni usar comandos de git para guardar cambios (git add, git commit, git push). El usuario lo hará de forma manual.

- Toda ventana de alerta o advertencia, requiera o no interaccion del usuario, debe ser de un formato Nativo de la app y no una notificacion de formato nativo del navegador.
